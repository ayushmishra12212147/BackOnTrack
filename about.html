About BackOntrack

BackOntrack is a lightweight educational website built with HTML, CSS, and vanilla JavaScript. It helps learners study unit-wise notes and practice questions without needing an account.

What we offer

  • Subjects: C Programming, DBMS, Java, and DSA
  • Six units per subject
  • Topic-wise notes with code blocks
  • Practice at the end of each unit: MCQs, theory, and coding prompts
